# AI Content Factory

## Overview
This project is an automated AI-based content creation and publishing system for YouTube Shorts and Instagram Reels.  
The system identifies trending topics, generates scripts, creates voice and video, and publishes content automatically twice a day.

## Goals
- Fully automated content pipeline
- 2 posts per day (morning & evening)
- Minimal manual intervention
- Low-cost, scalable system

## High-Level Flow
1. Trend identification
2. Script generation
3. Voice generation
4. Video creation
5. Caption & hashtag generation
6. Upload to platforms
7. Performance tracking & learning

## Tech Stack (Planned)
- Python
- Docker
- OpenAI API
- FFmpeg / MoviePy
- YouTube Data API
- Instagram Graph API
