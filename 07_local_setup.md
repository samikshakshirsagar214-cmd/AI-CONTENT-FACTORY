# Local Development Setup

This project will be developed and tested locally before moving to cloud deployment.

## Requirements
- Python 3.10+
- VS Code
- Git
- Docker
- 20–30 GB free storage
- Internet connection (for APIs)

## Setup Steps
1. Create Python virtual environment
2. Install dependencies using `requirements.txt`
3. Add API keys in `.env` file
4. Run Manager Agent locally

## Local Testing
- All agents run on local machine
- Outputs stored locally
- Auto-upload can be disabled
- No cloud or hosting cost during testing

## Why Local First
- Zero cost
- Faster debugging
- Safe experimentation
- Ready before production

## Next Step
Once stable, the same setup will be moved to cloud with scheduling and auto-posting enabled.
